# text decoration set
Aviutl2のテキスト装飾セット

テキストの簡単な設定をリストから選択できます。
数値や色は変更を行えません。

リスト名
赤,白,黒などはそのエフェクトの色
FはFrameで縁取り
HはHard Shadowでぼかしの数値が0のドロップシャドウ
SはSoft Shadowでぼかしの数値が0よりも大きなドロップシャドウ。主に10。
色+色は前後のエフェクトの色が違うだけで同じエフェクトであることを表す。

数値の記載はないが、実際には数値が違うFなどの例も多数存在する。
数値の違いにより別の表記としているのは、主にHSとなる。

黒F白H白F黒S
は
黒い縁取り
白いドロップシャドウ(ぼかしの数値が0)
白い縁取り
黒いドロップシャドウシャドウ(ぼかしの数値が1以上。主に10)

白+赤F黒H黒F黒S
は
白い縁取り
赤い縁取り
黒いドロップシャドウ(ぼかしの数値が0)
黒い縁取り
黒いドロップシャドウシャドウ(ぼかしの数値が1以上。主に10)

構造プリセットを選択した場合、数値や色を変更できます。

README.md
の2つ以外は、すべて動作に必要なので、すべて同じフォルダに入れてください。

既知の不具合:
文字を個別オブジェクトにして、グラデーションや縁取りの指定文字実行を使用すると非常に重くなる。私の環境では落ちた。
コードを書き換えて新たなプリセット項目を新しく登録すると、プロジェクトファイルの文字設定がリセットされる。

二次配布可、改変可、クレジット不要
